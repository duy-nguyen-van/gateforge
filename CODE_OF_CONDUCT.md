# Code of Conduct

## Our pledge

We are committed to a welcoming, harassment-free community for everyone — regardless of age, background, identity, or experience level.

## Expected behavior

- Be respectful and considerate in issues, pull requests, reviews, and discussions.
- Use inclusive language.
- Accept constructive criticism graciously.
- Focus on what is best for the project and its users.
- Assume good intent; disagree on technical merits, not people.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or sustained disruption.
- Publishing others' private information without permission.
- Sexualized language or imagery, or unwelcome sexual attention.

## Scope

This code applies to all project spaces: the GitHub repository (issues, pull requests, discussions, security advisories) and any community channels officially linked from this project.

## Enforcement

Maintainers may remove, edit, or reject contributions and temporarily or permanently ban contributors for behavior they deem inappropriate, threatening, offensive, or harmful.

To report a violation:

1. Open a **private** GitHub issue or contact a maintainer directly if you have an existing channel.
2. Include what happened, when, links or screenshots if available, and who was involved.

Reports are reviewed promptly and handled confidentially. Retaliation against reporters is not tolerated.
